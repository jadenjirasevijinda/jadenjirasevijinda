Setup Guide

How to add your photos & videos

Step-by-step — no coding required.

🗂️
Adding Photos
Put files in the images/ folder, then reference them in content.js.
  1. Compress your photo at squoosh.app (80% quality, save as JPG)
  2. Move the file into your images/ folder
  3. Open content.js and find the photos array
  4. Set src: "images/your-file.jpg"
  5. Run git add . && git commit -m "add photos" && git push
💡 For lots of photos, use cloudinary.com (free) and paste the URL instead of a local path.
▶️
Adding Videos
Upload to YouTube (can be Unlisted), then paste the link into content.js.
  1. Go to youtube.com and upload your video
  2. Set visibility to Unlisted if you want it private-ish
  3. Copy the video URL (e.g. youtube.com/watch?v=abc123)
  4. Open content.js and find the videos array
  5. Paste URL into the url field — thumbnail auto-loads!
  6. Push to GitHub
💡 Leave thumbnail: null and the YouTube thumbnail loads automatically.
content.js — example entries
// Photo example
{ src: "images/del-valle.jpg", category: "photography", caption: "Golden hour", featured: true },

// Video example
{ title: "Livermore Aerial 2024", url: "https://youtube.com/watch?v=YOUR_ID", category: "drone", thumbnail: null }
